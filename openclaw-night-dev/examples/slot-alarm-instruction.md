# 開発指示書: スロット風目覚ましアプリ

## 目的
GitHub Pages 上で動く、スロット風 UI の目覚まし Web アプリを作る。アラームを止めるには 3 桁のリールで `777` を揃える必要がある。

## スコープ
- 静的 HTML / CSS / JavaScript だけで動く
- GitHub Pages 互換
- アラーム時刻を 1 件設定できる
- 指定時刻になったら画面にアラーム状態を表示し、音または視覚的な強い通知を出す
- アラーム停止には 3 リールを回して `777` を揃える必要がある
- 各リールは停止時に 3 コマ程度のすべり目演出を入れる
- スマホ表示でも操作しやすい UI にする
- localStorage に設定を保存する

## 非スコープ
- バックエンド
- ユーザーアカウント
- 複数アラーム管理
- 外部 API
- ネイティブアプリ化

## 制約
- 既存 repo を壊さない
- 相対パスで動作する
- 追加ライブラリなしでも成立することを優先
- 音がブラウザ制約で鳴らない場合に備えて、視覚的な通知を必ず入れる

## 完了条件
- `slot-alarm/` ディレクトリ配下にアプリ一式が存在する
- `index.html`, `style.css`, `script.js` がある
- アラーム時刻を設定して保存できる
- 指定時刻でアラーム状態になる
- `777` が揃うまで解除できない
- すべり目演出が見える
- スマホ表示でも最低限遊べる
- README か説明文がある

## pushポリシー
- push してよい

## 不明点の扱い
- 妥当な仮定で進め、最後に仮定内容をまとめる

## 補足
- デザインは少しゲームっぽくてよい
- 目覚ましとして最低限わかりやすいことを優先
- 音が難しければバイブ風アニメーションや点滅を入れてよい
